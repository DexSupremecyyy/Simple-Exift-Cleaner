# PROJEK SIMPLE Exift Cleaner


Projek simple menggunakan python CLI untuk menghapus EXIFT metadata dari foto kita.
Dibuat menggunakan Python & Pillow


# Features
- Deteksi metadata tersembunyi
- Menghilangkan EXIFT data seperti (GPS, Kamera Model, dll)
- Support JPG/PNG

# Cara Install
1. Clone Reponya
2. `pip install Pillow`
3. `python cleaner.py`

# Cara Pakai
1. `python cleaner.py`
2. masukan nama file nya
3. ENTER
